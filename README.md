# 📚 OrBook

**"Leitores conectados por histórias."**

Bem-vindo(a) ao projeto **OrBook**! Este README apresenta informações sobre o projeto, suas funcionalidades, tecnologias utilizadas, forma de execução e como contribuir para seu desenvolvimento.

## Sobre o Projeto

O **OrBook** é uma plataforma social voltada exclusivamente para leitores, criada para conectar pessoas por meio da literatura. Inspirado no conceito das comunidades do Orkut, o sistema oferece um ambiente onde usuários podem criar, participar e administrar comunidades dedicadas aos mais diversos temas literários.

A proposta do projeto é transformar a leitura em uma experiência mais social, permitindo que os leitores encontrem pessoas com interesses semelhantes, compartilhem opiniões, debatam interpretações e descubram novas obras. Além das comunidades, o sistema conta com perfis altamente personalizáveis para que cada usuário expresse sua identidade literária, exibindo seus livros, autores e personagens favoritos.
